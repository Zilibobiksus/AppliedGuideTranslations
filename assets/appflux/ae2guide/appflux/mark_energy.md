# Как пометить FE в слоте конфигурации

Вам может понадобиться вывести энергию с помощью <ItemLink id="ae2:export_bus"/>, и для этого нужно пометить FE в его слоте конфигурации.

Щёлкните правой кнопкой мыши по слоту энергоносителем, чтобы пометить FE.
