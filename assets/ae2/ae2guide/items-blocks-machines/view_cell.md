# Ячейка просмотра

<ItemImage id="view_cell" scale="2" />

Ячейки просмотра используются для фильтрации отображения [терминалов](terminals.md). Вы разделяете их в <ItemLink id="cell_workbench" />.

Например, предположим, вы хотите, чтобы терминал отображал только выбор строительных материалов из камня. Разделите ячейку просмотра на эти материалы и поместите её в терминал, и будут показаны только эти предметы.

Ячейки просмотра являются аддитивными: если у вас есть ячейка просмотра для дубовых досок и ячейка просмотра для булыжника, помещение обеих покажет и доски, и булыжник.
