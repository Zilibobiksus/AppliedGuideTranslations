# Кабельный якорь

<GameScene zoom="6" background="transparent">
  <ImportStructure src="../assets/assemblies/cable_anchor.snbt" />
  <IsometricCamera yaw="195" pitch="30" />
</GameScene>

Небольшие декоративные шипы, устанавливаемые на кабели, которые можно использовать, чтобы предотвратить соединение кабелей, создать лестницы из кабелей или заставить кабель выглядеть так, будто он подключён к окружающим стенам. Также используется для создания <ItemLink id="facade" />.

Кабельные якоря предотвращают образование соединений с той стороны, на которой они установлены.

По ним можно лазать, как по лестнице.
