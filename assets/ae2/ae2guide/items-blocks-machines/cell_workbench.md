# Верстак для ячеек

<BlockImage id="cell_workbench" scale="8" />

Верстак для ячеек позволяет настраивать [ячейки хранения](storage_cells.md) и <ItemLink id="view_cell" />.

Вы можете добавлять [карты улучшений](upgrade_cards.md) и настраивать «разделы» для фильтрации того, что ячейка будет хранить.

Предметы и жидкости можно перетаскивать в слоты из JEI/REI, даже если у вас на самом деле нет ни одного из этих предметов.

Щёлкните правой кнопкой мыши по контейнеру с жидкостью (например, по ведру или резервуару), чтобы установить эту жидкость в качестве фильтра вместо предмета ведра или резервуара.

## Настройки

В верстаке для ячеек есть несколько кнопок в верхнем левом углу:

*   Вы можете разделить ячейку на основе её существующего содержимого
*   Вы можете очистить разделение ячейки
*   Верстак можно настроить так, чтобы он сохранял настройки разделения при удалении ячейки, что позволяет копировать настройки между ячейками.
