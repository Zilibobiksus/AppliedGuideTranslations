# Аппликатор цвета

<ItemImage id="color_applicator" scale="4" />

Аппликатор цвета используется для покраски окрашиваемых блоков, таких как [кабели](cables.md), шерсть, терракота, стекло и бетон. Он использует [шарики с краской](paintballs.md) или красители, а снежки можно использовать для удаления цвета с кабелей и следов от шариков с краской с блоков.

Его энергию можно восстановить в <ItemLink id="charger" />.

Аппликаторы цвета действуют как [ячейки хранения](storage_cells.md), и их хранилище краски можно наиболее легко заполнить, поместив аппликатор в слот для ячейки хранения в <ItemLink id="chest" />.

Щёлкните правой кнопкой мыши по цели, чтобы применить выбранный цвет. Чтобы изменить цвета, используйте колесо прокрутки, удерживая клавишу Shift, или щёлкните правой кнопкой мыши без цели.

## Улучшения

Аппликаторы цвета поддерживают следующие [улучшения](upgrade_cards.md), которые вставляются через <ItemLink id="cell_workbench" />:

*   <ItemLink id="equal_distribution_card" /> выделяет одинаковое количество места в байтах ячейки для каждого типа, чтобы один тип не мог заполнить всю ячейку
*   <ItemLink id="void_card" /> уничтожает вставляемые предметы, если ячейка заполнена (или выделенное для этого конкретного типа место в случае карты равномерного распределения). Будьте осторожны с разделением этого!
*   <ItemLink id="energy_card" /> для увеличения ёмкости их аккумулятора
