# Пространственный якорь

<BlockImage id="spatial_anchor" p:powered="true" scale="8"/>

Для функционирования сети AE2 необходимо загружать чанки, где находятся её [устройства](../ae2-mechanics/devices.md). Если загружена только часть сети, она может работать некорректно. Пространственный якорь решает эту проблему, принудительно загружая чанки, в которых находится сеть. Достаточно одного кабеля, пересекающего границу чанка, чтобы загрузить новый чанк.

Якорь распространяет загрузку через [квантовые мостики](quantum_bridge.md), но не сквозь измерения. Если у вас есть квантовый мостик в Нижний мир, вам понадобится пространственный якорь в сети вашей базы и в сети Нижнего мира.

По умолчанию он также включает случайные тики в загружаемых чанках. Это можно отключить в настройках AE2.

Его можно повернуть с помощью <ItemLink id="certus_quartz_wrench" />, если по какой-то причине вам это нужно.

## Настройки

*   Пространственный якорь предоставляет доступ к глобальной настройке для просмотра энергии в AE или E/FE.
*   Можно отобразить голографическое изображение в мире, показывающее загружаемые чанки.

## Энергия

Пространственный якорь будет потреблять [энергию](../ae2-mechanics/energy.md) согласно уравнению:

e = 80 + (x\*(x+1))/2

где x — количество загружаемых чанков
