# Карта памяти

<ItemImage id="memory_card" scale="4" />

Карта памяти используется для копирования и вставки настроек между [устройствами](../ae2-mechanics/devices.md) AE2 и для связывания [тоннелей точка-точка](p2p_tunnels.md). Она также может вставлять [карты улучшений](upgrade_cards.md) в устройства.

- Щёлкните правой кнопкой мыши с зажатым Shift, чтобы скопировать настройки или сгенерировать новую частоту связи P2P.
- Щёлкните правой кнопкой мыши, чтобы вставить настройки, карты улучшений или частоту связи.
