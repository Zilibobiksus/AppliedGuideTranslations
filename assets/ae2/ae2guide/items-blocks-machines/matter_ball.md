# Шарики материи

<ItemImage id="matter_ball" scale="4" />

Шарик из общей материи, полезен как боеприпас для <ItemLink id="matter_cannon" /> или для производства [шариков с краской](paintballs.md).

Создаётся из 256 предметов или вёдер в <ItemLink id="condenser" /> в режиме шара материи.
