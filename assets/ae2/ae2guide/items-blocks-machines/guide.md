# Руководство

<ItemImage id="guide" scale="8" />

### Это руководство, которое вы сейчас читаете, для всех ваших потребностей в руководстве по AE2.

* Доступ к боковой панели слева, чтобы найти оглавление
* На многих страницах есть интерактивные сцены. Если рядом со сценой есть кнопки ![Плюс](../assets/diagrams/plus.png) и ![Минус](../assets/diagrams/minus.png) (масштабирование), вы можете вращать и перемещать камеру. Щелкните левой кнопкой мыши и перетащите, чтобы вращать, щелкните правой кнопкой мыши и перетащите, чтобы перемещать.
