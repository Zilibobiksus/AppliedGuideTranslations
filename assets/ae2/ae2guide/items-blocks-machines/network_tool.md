# Сетевой инструмент

<ItemImage id="network_tool" scale="4" />

Сетевой инструмент — это модифицированный [гаечный ключ](wrench.md), который также показывает диагностическую информацию о сети и может хранить [карты улучшений](upgrade_cards.md). Хотя он сохраняет способность гаечного ключа быстро разбирать вещи и снимать [подчасти](../ae2-mechanics/cable-subparts.md) с кабеля, он не может поворачивать вещи.


У него есть 9 слотов для хранения [карт улучшений](upgrade_cards.md), и они будут доступны в любом графическом интерфейсе устройства AE2, если инструмент находится где-либо в вашем инвентаре.


Щелчок правой кнопкой мыши по любой части сети покажет окно диагностической информации, аналогичное щелчку правой кнопкой мыши по <ItemLink id="controller" />. Это окно показывает:


*   Количество используемых каналов в сети
*   Переключатель глобальной настройки для просмотра энергии в AE или E/FE
*   Количество [энергии](../ae2-mechanics/energy.md), хранящейся в сети, и максимальную ёмкость энергии сети
*   Количество энергии, поступающей в сеть и используемой сетью
*   Список всех [устройств](../ae2-mechanics/devices.md) и компонентов в сети

Это окно также полезно для определения того, являются ли два разных кабеля или устройства частью одной сети при работе с [подсетями](../ae2-mechanics/subnetworks.md).


## Скрытие фасадов

<a href="facades.md">Фасады</a> будут скрыты при удерживании сетевого инструмента в любой руке.


Вы можете взаимодействовать с блоками за скрытыми фасадами, не снимая фасады.
