# Беспроводные терминалы

<Row>
  <ItemImage id="wireless_terminal" scale="4" />

  <ItemImage id="wireless_crafting_terminal" scale="4" />
</Row>

Беспроводные терминалы — это портативные версии обычных проводных [терминалов](terminals.md). У них точно такие же пользовательские интерфейсы, как и у их проводных аналогов, за исключением того, что вместо слотов для <ItemLink id="view_cell" /> они имеют слоты для [карт улучшений](upgrade_cards.md)

Для сопряжения с сетью вставьте терминал в правый верхний слот <ItemLink id="wireless_access_point" />, подключённого к этой сети. (Слот с изображением беспроводного терминала и стрелкой под ним)

Они должны находиться в пределах досягаемости <ItemLink id="wireless_access_point" />, чтобы функционировать.

Их энергию можно восстановить в <ItemLink id="charger" />.

# Беспроводной терминал

<ItemImage id="wireless_terminal" scale="4" />

Ваш базовый терминал, теперь портативный! Просматривайте и получайте доступ к содержимому [сетевого хранилища](../ae2-mechanics/import-export-storage.md) и запрашивайте вещи из своей настройки [автосоздания](../ae2-mechanics/autocrafting.md) из любого места в пределах досягаемости <ItemLink id="wireless_access_point" />.

# Беспроводной терминал изготовления

<ItemImage id="wireless_crafting_terminal" scale="4" />

Беспроводной терминал изготовления похож на обычный беспроводной терминал, со всеми теми же настройками и разделами, но с добавленной сеткой крафта, которая будет автоматически пополняться из [сетевого хранилища](../ae2-mechanics/import-export-storage.md). Будьте осторожны при использовании Shift+щелчка по результату!
