# Универсальный беспроводной терминал

<ItemImage id="ae2wtlib:wireless_universal_terminal" scale="3" />

<ItemLink id="ae2wtlib:wireless_universal_terminal" /> — это объединение нескольких [беспроводных терминалов](wireless_terminals.md) в один предмет.

Все [беспроводные терминалы AE2WTLib](wireless_terminals.md) могут быть объединены в один терминал.

<ItemLink id="ae2:wireless_terminal" /> из AE2 ***НЕЛЬЗЯ*** объединить в <ItemLink id="ae2wtlib:wireless_universal_terminal" />.

## Интерфейс

<ItemLink id="ae2wtlib:wireless_universal_terminal" /> добавляет кнопку в интерфейс терминала, нажатие на которую левой (или правой) кнопкой мыши позволяет переключаться на следующий (или предыдущий) установленный [беспроводной терминал](wireless_terminals.md).

## Улучшения

<ItemLink id="ae2wtlib:wireless_universal_terminal" /> принимает те же улучшения, что и установленные в него [беспроводные терминалы](wireless_terminals.md). Каждый установленный терминал добавляет 2 слота для улучшений.

## Хранение энергии

При добавлении терминалов их ёмкость аккумулятора суммируется. Таким образом, <ItemLink id="ae2wtlib:wireless_universal_terminal" /> с двумя установленными терминалами имеет вдвое больший запас энергии, чем один [беспроводной терминал](wireless_terminals.md).
